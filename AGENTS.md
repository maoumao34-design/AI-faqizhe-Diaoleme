# Agent Instructions

## Design Reference

Before changing the Diaoleme UI, review the original design references in `docs/design-reference/`.

- Start with `docs/design-reference/README.md` to understand which image maps to each page.
- Use `docs/design-reference/images/` as the visual source of truth for layout, mood, colors, component grouping, and page intent.
- Prefer matching the design direction before introducing new visual patterns.
- If implementation differs from the reference, mention the reason in the PR or task summary.
